# 要件定義
■カートページ(cart.jsp) 
○カートの中身を確認するページ
・カートに入っている商品の一覧、金額の表示
・商品ID、価格、小計
・合計金額表示
・ラッピングの有無
・支払方法

■注文確認ページ(cartbuyconfirm.jsp)
〇注文内容の確認
・カートに入っている商品の一覧、金額の表示
・商品ID、価格、小計
・合計金額表示
・ラッピングの有無
・支払方法

■注文リザルトページ(cartfinish.jsp)
○注文確定のページ
・カートページ(cartbuyconfirm.jsp)から移動できる
・「ご注文、ありがとうございました」表示
・到着予定日の表示
・トップページ(index.html)へのリンク

以上の動作をMVCモデルで作成しました。
